Passive Guard Simulation

A local JavaScript demo that mimics the shape of passive bot detection: collect browser interaction signals, produce a risk score, and send the verdict with the form.

Behavior signals

Pointer movement, keyboard use, field focus, time on page, scrolling, and tab visibility feed the local score.

Silent challenge

No visible puzzle is shown. The visitor receives a low-risk, review, or high-risk verdict when submitting.

Server gate

A production version would combine this with a signed challenge, rate limits, and server-side verification.
